# Slag

Slag is a tool for generating new Ruby gems. It is in heavy development and isn't stable yet.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'slag'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install slag
